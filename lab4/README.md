## 🎯 Мета роботи
Ознайомитися з принципами роботи регістрів зсуву (на прикладі мікросхеми 74HC595) та семисегментних світлодіодних індикаторів у мікроконтролерних системах на базі PSoC. Набути практичних навичок створення проєктів у інтегрованому середовищі PSoC® Creator™, налаштування апаратних компонентів для керування індикаторами через зсувні регістри та програмної реалізації динамічної індикації.

## 🛠 Хід виконання роботи

### 1. Підготовка середовища та проєкту
* За основу було взято напрацювання з попередньої роботи (матрична клавіатура) для інтеграції вводу та виводу даних.
* До схеми проєкту (`TopDesign.cysch`) додано компоненти для керування зсувним регістром, який, у свою чергу, керує 4-розрядним 7-сегментним індикатором.
* Налаштовано необхідні цифрові виходи (Digital Output Pins) для ліній даних (Data), тактування (Clock) та засувки (Latch) зсувного регістру 74HC595.

### 2. Програмна реалізація керування індикатором
* Підключено бібліотеку/функції для роботи з індикатором через зсувний регістр (наприклад, `FourDigit74HC595_sendOneDigit`).
* Налаштовано логіку перетворення введених символів у відповідні коди для 7-сегментного дисплея (включаючи символи від '0' до '9', літери 'A'-'F' та спеціальні символи, як-от тире).
* Реалізовано антибрязкіт (Debounce) для кнопок матричної клавіатури за допомогою затримки `CyDelay(50)`.

## 🧑‍💻 Інтеграція клавіатури та 7-сегментного індикатора

У рамках роботи було реалізовано алгоритм, який зчитує натискання з матричної клавіатури та виводить відповідний символ на 7-сегментний індикатор:

1. **Цифровий режим:** При натисканні клавіш від `0` до `9` на дисплеї відображається відповідна цифра.
2. **Спеціальні символи:** При натисканні клавіші `#` (індекс 11) на дисплеї відображається тире (`-`).
3. **Символьний (шістнадцятковий) режим:** Інші кнопки налаштовано на виведення символів шістнадцяткової системи числення (`A`, `b`, `C`, `d`, `E`, `F`).

### Фрагмент коду обробки натискань (`main.c`):
```c
// Якщо натиснута кнопка від 0 до 9
if (current_pressed <= 9) {
    FourDigit74HC595_sendOneDigit(0, current_pressed, 0);
} 
// Якщо натиснута кнопка '#' (індекс 11) - відображаємо тире
else if (current_pressed == 11) {
    FourDigit74HC595_sendOneDigit(0, 10, 0); // 10 = індекс тире
} 
// Режим літер (A, b, C, d, E, F)
else {
    switch (current_pressed) {
        case 1:  FourDigit74HC595_sendOneDigit(0, 11, 0); break; // A
        case 2:  FourDigit74HC595_sendOneDigit(0, 12, 0); break; // b
        case 3:  FourDigit74HC595_sendOneDigit(0, 13, 0); break; // C
        case 4:  FourDigit74HC595_sendOneDigit(0, 14, 0); break; // d
        case 5:  FourDigit74HC595_sendOneDigit(0, 15, 0); break; // E
        case 6:  FourDigit74HC595_sendOneDigit(0, 16, 0); break; // F
        default: FourDigit74HC595_sendOneDigit(0, 10, 0); break; // '-'
    }
}

CyDelay(50); // Затримка для антибрязкоту
```

## 📝 Висновок
Під час виконання лабораторної роботи було успішно ознайомлено з принципами керування семисегментними індикаторами за допомогою зсувних регістрів 74HC595, що дозволяє значно економити кількість використовуваних виводів (пінів) мікроконтролера. Набуто практичних навичок інтеграції кількох периферійних модулів (матрична клавіатура на вхід та 7-сегментний дисплей на вихід) у єдину систему в середовищі PSoC Creator. Написано та відлагоджено програмний код мовою С для коректної трансляції натискань клавіш у відповідні візуальні символи на дисплеї.
