## 🛠 Хід виконання роботи

### 1. Підготовка середовища та проєкту
* Встановлено термінальну програму **PuTTY** для моніторингу послідовного порту.
* За основу взято та скопійовано проєкт з Лабораторної роботи №1.
* Модифіковано схему в `TopDesign.cysch`: додано три окремі компоненти `Digital Output` для кожного каналу RGB-світлодіода (`LED_R`, `LED_G`, `LED_B`).
  * Drive mode: `Strong drive`
  * Initial drive state: `High (1)` (стан вимкнено).

### 2. Налаштування UART та пінів
* З каталогу компонентів (`Communications`) додано модуль **Software Transmit UART**.
* **Конфігурація UART:** встановлено Baud Rate на `115200` біт/с.
* **Призначення пінів (Pin Mapping):**
  * `LED_R` — P1.6
  * `LED_G` — P0.2
  * `LED_B` — P0.3
  * `\SW_Tx_UART:tx\` — P2.7
* Налаштовано апаратні перемикачі на платі (SW4.1 та SW4.2 в положення OFF, SW4.3 — ON) для підключення виводу PSoС до входу програматора.

### 3. Програмна реалізація та тестування
* Написано код мовою C для виводу інформації про стан кнопки через інтерфейс UART у термінал комп'ютера.
* Проєкт скомпільовано та завантажено (запрограмовано) у мікроконтролер плати **CY8CKIT-042 PSoC 4 Pioneer Kit**.
* У програмі PuTTY налаштовано підключення до відповідного COM-порту мікроконтролера. 
* В результаті роботи у термінал успішно виводяться повідомлення: `Button pressed` (кнопка натиснута) та `Button released` (кнопка відпущена).

## 🧑‍💻 Самостійне завдання (Варіант 13)

У рамках індивідуального завдання було реалізовано наступне:

1. **Ідентифікація:** Додано вивід власного імені та прізвища у консоль (`Yaroslav Senyk - Variant 13`).
2. **Колірна модель:** Досліджено та заповнено таблицю керування RGB-світлодіодом за допомогою логічних рівнів (0 - світиться, 1 - не світиться) для різних кольорів.
3. **Модифікація логіки миготіння:** Створено програму, яка динамічно змінює колір блимання світлодіода (з інтервалом у 500 мс) та надсилає стан через UART залежно від положення кнопки:
   * **Кнопка натиснута:** світлодіод блимає Блакитним (Cyan) кольором, у термінал виводиться `Button Pressed: Blinking Cyan`.
   * **Кнопка відпущена:** світлодіод блимає Зеленим (Green) кольором, у термінал виводиться `Button Released: Blinking Green`.

### Фрагмент коду індивідуального завдання (`main.c`):
```c
#include "project.h"

int main(void)
{
    CyGlobalIntEnable; /* Enable global interrupts. */
    
    // Ініціалізація UART та вивід імені
    SW_TX_UART_Start();
    SW_TX_UART_PutString("Yaroslav Senyk - Variant 13\r\n");

    for(;;)
    {
        // 0 - кнопка натиснута, 1 - відпущена
        if (Button_Read() == 0) 
        {
            SW_TX_UART_PutString("Button Pressed: Blinking Cyan\r\n");
            
            // Вмикаємо Cyan (Зелений + Синій)
            LED_R_Write(1); 
            LED_G_Write(0); 
            LED_B_Write(0); 
            CyDelay(500);
            
            // Вимикаємо всі
            LED_R_Write(1);
            LED_G_Write(1);
            LED_B_Write(1);
            CyDelay(500);
        }
        else 
        {
            SW_TX_UART_PutString("Button Released: Blinking Green\r\n");
            
            // Вмикаємо Green
            LED_R_Write(1); 
            LED_G_Write(0); 
            LED_B_Write(1); 
            CyDelay(500);
            
            // Вимикаємо всі
            LED_R_Write(1);
            LED_G_Write(1);
            LED_B_Write(1);
            CyDelay(500);
        }
    }
}
```
