## 🎯 Мета роботи
Ознайомитися з принципами підключення та використання матричної клавіатури для введення інформації у мікроконтролерних системах на базі PSoC. Набути навичок розробки проєкту в інтегрованому середовищі PSoC® Creator™, налаштування апаратних компонентів та програмної обробки введених даних.

---

## 🛠 Хід виконання роботи

### 1. Підготовка середовища та проєкту
* За основу взято та скопійовано проєкт з Лабораторної роботи №2.
* Модифіковано схему в `TopDesign.cysch`:
  * З каталогу `Cypress` додано 3 двонаправлені піни (`Digital Bidirectional Pin`) для стовпців матриці (Columns).
  * Додано 4 піни типу «Вхід» (`Digital Input Pin`) для рядків матриці (Rows).

### 2. Налаштування пінів
* **Конфігурація стовпців (Columns):** `Drive mode` встановлено як `Open drain, drives low`, `Initial drive state` — `High (1)`.
* **Конфігурація рядків (Rows):** `Drive mode` встановлено як `High impedance digital`, `Initial drive state` — `Low (0)`.
* **Призначення пінів (Pin Mapping):** Порти введення-виведення налаштовано відповідно до схеми Шилда для плати PSoC 4 Pioneer Kit. Перемикачі апаратного блоку `SW3` переведено в положення `ON` для активації матричної клавіатури (замість використання слайдера CapSense).

### 3. Програмна реалізація та тестування
* Створено масиви вказівників на функції для зручної конфігурації, запису та читання станів пінів.
* Написано власні функції для ініціалізації (`initMatrix`) та зчитування станів (`readMatrix`) матричної клавіатури.
* Додано вивід натиснутих та відпущених кнопок у термінал через UART (події `Button X pressed` та `Button X released`).

## 🧑‍💻 Самостійне завдання 

У рамках самостійного/індивідуального завдання було реалізовано систему авторизації та керування кольорами:

1. **Система паролів:** Встановлено 4-значний пароль (`1234`). При старті система заблокована.
   * У разі успішного введення в термінал виводиться `Access allowed! System Unlocked.`
   * При помилковому введенні — `Access denied! Try again:`.
2. **Керування кольорами (через конструкцію `switch(case)`):** Після розблокування системи натискання різних клавіш на матричній клавіатурі змінює колір RGB-світлодіода (Червоний, Зелений, Синій, Жовтий, Фіолетовий, Блакитний). 
3. **Алгоритм світіння:** Якщо кнопку натиснуто і утримується — світлодіод світиться відповідним кольором. Якщо кнопку відпущено — світлодіод вимикається (Black). На старті відразу після розблокування горить колір White.

### Фрагмент коду логіки авторизації (`main.c`):
```c
// Режим вводу пароля
input_pass[pass_index] = current_pressed;
pass_index++;
SW_Tx_UART_PutString("*");

if(pass_index == 4) 
{
    SW_Tx_UART_PutCRLF();
    
    // Перевірка еталонного пароля (1-2-3-4)
    if(input_pass[0] == correct_pass[0] && input_pass[1] == correct_pass[1] && 
       input_pass[2] == correct_pass[2] && input_pass[3] == correct_pass[3]) 
    {
        SW_Tx_UART_PutString("Access allowed! System Unlocked.");
        is_logged_in = 1; // РОЗБЛОКОВУЄМО СИСТЕМУ
        
        // Згідно з завданням, після розблокування горить White
        LED_R_Write(0); 
        LED_G_Write(0); 
        LED_B_Write(0);
    } 
    else 
    {
        SW_Tx_UART_PutString("Access denied! Try again:");
    }
    SW_Tx_UART_PutCRLF();
    pass_index = 0; // Скидаємо лічильник для наступної спроби
}
```
